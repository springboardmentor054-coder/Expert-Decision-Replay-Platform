# Features

The Expert Decision Replay Platform is designed to capture, organize, approve, and reuse organizational decisions while preserving institutional knowledge.


# 1. Authentication

Provides secure access to the platform.

### Features
- User Registration
- Secure Login
- JWT Authentication
- Password Encryption
- Password Reset
- Session Management
- Account Verification
- Role-Based Access Control (RBAC)



# 2. User Management

Provides complete user administration and profile management.

### Features

- Create Users
- View User Profiles
- Update User Information
- Delete Users
- Activate/Deactivate Users
- Assign Roles
- Manage Departments
- Search Users
- Filter Users
- Profile Management
- Change Password
- Upload Profile Picture

### User Roles

#### Administrator

- Manage Users
- Assign Roles
- View Activity Logs
- Manage Permissions

#### Reviewer

- Review Decisions
- Approve or Reject Decisions
- Provide Feedback

#### Employee

- Submit Decisions
- Update Profile
- Track Decision Status
